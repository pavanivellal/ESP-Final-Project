# Estatecart
E-Commerce application for real estate industry

The application has an efficient inventory management system, customizable payment schemes, well defined roles, email notification system, user friendly interface. It is developed on a PHP server with MYSQL for Database. The front end of the application is developed using HTML, Bootstrap and Javascript.
